# 7.-Temperature-Converter-Celsius-Fahrenheit-
Objective:
To convert temperature from Celsius to Fahrenheit using a servlet without JDBC.
Description:
User enters a temperature in Celsius through an HTML form.
The servlet reads the value in the service() method, performs the conversion:
F = (C * 9/5) + 32
and displays the Fahrenheit output on the browser.
Requirements:
HTML:
Input field for Celsius
Submit button
Servlet:
Get Celsius value
Perform conversion
Print formatted output
then connect jar and apache tomcat 10.1 and run on server.
